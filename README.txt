New Voodoo Portfolio — Rev 3

Contents included:
- index.html
- styles.css
- script.js
- assets/images (hero/background, gallery set, NVD cube art)
- assets/audio (5 MP3 tracks)
- assets/video/page-loader.mp4
- assets/thumbs (7 skull thumbnails for YouTube links)

Deployment:
1. Unzip the package.
2. Drag the folder contents into Netlify, or connect the folder to your preferred static deployment flow.
3. No build step is required.

Notes:
- Hero uses home-bg.png as the first visual.
- Video cards open YouTube in a new tab.
- Gallery images open in a built-in lightbox.
- Ambient particles and drifting cubes are layered above the page.
